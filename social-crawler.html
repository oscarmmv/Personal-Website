Social Crawler

Social crawler is a tool that can be setup to run server side to track a users digital footprint.


I was inspired by the sherlock project to create a tool to retrieve metrics from a users social media account. This project scrapes user information such as followers, following, likes, posts, bio, location, and age, without the use of API. Social crawler utilizes Selenium, Helium, and HTTP GET requests to retrieve user information from Facebook, Instagram, TikTok, and Twitter.
