Mouse Heatmap

The mouse heatmap is a tool that can be used to track critical information to visualize what users are interacting with on your page to better understand engagement of site.


This lightweight application can be setup to be run server side. It stores an array of mouse points and plays a heatmap of the user's mouse movements. Additionally, it returns a JSON object including the mouse points, clicks, and time on and away from page.
